# 상용 OCR API 총정리 비교

## 1. Naver CLOVA OCR

### 개요
네이버 클라우드에서 제공하는 상용 OCR 서비스로, 한국어를 포함한 다양한 언어 지원과 문서 특화 템플릿 OCR을 제공한다.

### 특징
- 한국어 최적화 (등기부등본, 사업자등록증 등 국내 문서 템플릿 제공)
- 언어 지원: 한국어, 영어, 일본어, 중국어 등
- 이미지/PDF 업로드 지원
- REST API 형태로 연동
- 번역, 말풍선 등 특화 기능
- 문서 형태별(인쇄, 수기, 표, 영수증) 분류 기능

### 가격
- 정확한 요금은 상업 요금제 또는 사용량과 협의 필요
- 네이버 클라우드 콘솔에서 단가 확인 가능
- 용도(일반 문서 / 템플릿 기반)에 따라 달라질 수 있음

### 장단점
+ 한국어 템플릿 지원이 강력함 (등기부등본 등에 적합한 후보)
+ 국내 업체로 법적 이슈 상대적으로 적음
+ 번역/문서 구조화 부가 기능 존재
- 가격 정보 공개가 제한적
- 대량 문서 처리 시 비용이 빠르게 증가 가능성

## 2. Google Cloud Vision API OCR

### 개요
Google Cloud 에서 제공하는 이미지/문서 분석 OCR 서비스로, 글로벌 사용량과 안정성이 높다.

### 특징
- DOCUMENT_TEXT_DETECTION 정확도가 준수함 (한국어 포함)
- 50개 이상 언어 지원
- PDF/이미지 모두 처리 가능
- 글로벌 서비스 (SLA 높은 편)
- 신규 고객 $300 무료 크레딧 제공

### 가격
- 매월 첫 1,000건 무료
- Text Detection / Document Text Detection: $1.50 / 1,000건
- 500만건 초과: $0.60 / 1,000건
- 여러 Feature 조합 시 각각 과금됨

### 장단점
+ 글로벌 성능 검증, 가격 투명성
+ 무료 크레딧으로 PoC 진행 가능
+ 한국어 인식률 준수
- 미국 법 이슈 대비 필요 가능성
- 임베딩/LLM 등과 연계하려면 별도 연동 필요

---

## 3. 비교 핵심

| 항목 | Naver CLOVA OCR | Google Cloud Vision OCR |
|------|----------------|------------------------|
| 한국어 표준 문서 OCR | 강력 | 준수 |
| 템플릿 기반 등기 OCR | 있음 (국내 특화) | 없음 (일반 OCR) |
| 가격 구조 | 문의 필요 | 명확 (종량제) |
| 무료 크레딧 | 제한적 검토 필요 | $300 제공 |
| 글로벌 안정성 | 국내 중심 | 높음 |
| 법적 이슈 | 상대적 안정 | 미국 클라우드 검토 필요 |
| 연동 난이도 | REST API | REST API |

## 4. budongsan PDF와의 연관성

- 핵심은 **등기부등본 PDF -> 구조화된 텍스트**
- CID-font / 이미지 PDF 문제를 상용 OCR로 대체 가능성 검토
- CLOVA의 템플릿 OCR이 업무 형식과fit할 가능성이 높음
- PoC는 Google $300 크레딧으로 먼저 가능
